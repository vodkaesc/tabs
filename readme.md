## a simple configurable new tab extension


</div>

## info

tabs is an extension that offers a modern new tab page, prioritizing customizability such as choosing wallpapers, bookmarks, and more. Additionally, it emphasizes a smooth user experience through its animations on the new page.


## features

-  Fast & Lightweight
-  Fully Customizable
-  Privacy-Focused
-  Smooth Animations
-  Smart Search Bar (compatible with any search engine)
-  Personalized Greeting
-  Bookmarks
-  Custom Wallpapers
-  Weather
-  Custom CSS Styling
-  Custom Favicon
-  Quick Config Sharing
-  Custom Fonts
